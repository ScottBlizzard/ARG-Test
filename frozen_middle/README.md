# Frozen Middle Baseline

This directory preserves the middle-phase baseline so final-project work can continue in the same repository without losing the earlier submission context.

Contents:

- `external_materials/`: files that originally lived beside the repository in the old `Middle` workspace, including proposal/report PDFs, zip packages, and other course materials.
- `repo_submission_package/`: the middle-phase submission package that originally lived inside the repository.

Guideline:

- Treat these materials as frozen reference artifacts.
- Do not edit them unless a later cleanup explicitly requires it.
- Build final-project deliverables outside this directory.
