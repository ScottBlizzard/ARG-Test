Requirement ID: currency_conversion_rate_expiry
Domain: e-commerce pricing

Rules:
1. An exchange_rate has an updated_at timestamp.
2. A rate is considered fresh when age_minutes is less than 30.
3. If the cached rate is stale (age_minutes at least 30), the system fetches a new rate.
4. If fetching a new rate fails, checkout is blocked with an error.
5. If target currency is unsupported, the system returns an error.
