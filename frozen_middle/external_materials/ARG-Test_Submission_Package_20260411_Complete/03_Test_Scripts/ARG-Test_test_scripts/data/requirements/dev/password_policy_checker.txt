Requirement ID: password_policy_checker
Domain: account security

Rules:
1. password is required and must be 8 to 20 characters.
2. password must contain at least one uppercase letter, one lowercase letter, one digit, and one special character from ! @ # $ %.
3. password must not contain whitespace.
4. password must not contain the username as a substring.
5. If the password is invalid, the system returns a validation error and stores nothing.
6. If the password is valid, the system accepts it and stores the hashed password.
