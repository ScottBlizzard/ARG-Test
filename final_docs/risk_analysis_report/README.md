# Risk Analysis Report Package

Files in this package:

- `02_risk_analysis_report_cn.md`: source document
- `02_risk_analysis_report_cn.pdf`: rendered submission PDF
- `figures/`: local figures used by the report
- `generate_package.py`: regenerate local figures and rebuild the PDF

Command:

```powershell
python final_docs\risk_analysis_report\generate_package.py
```
